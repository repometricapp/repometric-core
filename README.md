# repometric-core

The open-source core application.

## Project Timeline

Repometric is being built in clear phases, starting with an open-source diagnostic core and evolving
toward more advanced use cases.

### Phase 1 — Single User / Organization (Public Repos Only)

- Support one GitHub user or organization
- Analyze public repositories only
- No authentication required
- Read-only GitHub API usage
- Open-source, local-first dashboard

### Phase 2 — Multiple Organizations (Public Repos Only)

- Support multiple users and/or organizations
- Public repositories only
- Aggregate and compare repository health
- No login required

### Phase 3 — Public + Private Repositories

- Support public and private repositories
- Requires GitHub authentication
- Foundation for a production-ready, hosted, or paid offering
- Built on top of this open-source core

### Phase 4 - Single User Privet Repos

- Support and private repositories
